up ()
// This function lets you move up multiple parent directories at once, so "up 3" is the same as "cd ../../..". It takes the number passed in as $1 and stores it in limit, then uses a for loop that runs limit times, appending "/.." to a variable d on each pass. After the loop, it pipes d through sed to strip the leading slash. If d ends up empty (meaning limit was 0 or blank), it defaults d to "..". Finally it cds into whatever path was built.
